## 乱拳打死老师傅

这个系列适合非GIS专业开发人员，这部分同学通常精通计算机开发，但对GIS可能不是那么了解。在工作中有时会涉及地图相关的工作，但公司又没有专人负责GIS开发，只能自己硬着头皮上了。百度地图API应该听过，但是不是用这百度地图API就能解决所有问题呢，这个系列或许能对你有所帮助。
